# محاسبه‌گر مصرف چرم

این پروژه یک اپ ساده اندروید برای ذخیره محصولات، محاسبه مابه‌التفاوت مصرف چرم و نگهداری سوابق ماهانه است.

## فرمول محاسبه

```text
مصرف استاندارد کل = تعداد × مصرف پیش‌فرض محصول
مابه‌التفاوت = چرم مصرفی واقعی − مصرف استاندارد کل
```

- اگر مابه‌التفاوت منفی باشد: **برش خوب**
- اگر مابه‌التفاوت مثبت باشد: **برش بد**
- اگر مابه‌التفاوت صفر باشد: **برش بدون اختلاف**

## امکانات

- افزودن و حذف محصول با مصرف پیش‌فرض چرم
- انتخاب ماه شمسی هنگام محاسبه مابه‌التفاوت
- جستجو و انتخاب محصول از بین محصولات ثبت‌شده
- ذخیره خودکار هر محاسبه در سوابق ماه انتخاب‌شده
- صفحه مستقل سوابق ماهانه با امکان فیلتر بر اساس ماه و جستجوی محصول
- حذف تکی سابقه با نگه‌داشتن انگشت روی آن و حذف همه سوابق
- ذخیره محصولات و سوابق روی خود گوشی با `SharedPreferences`
- پشتیبانی از اعداد فارسی و انگلیسی

## اجرای پروژه در Android Studio

1. پروژه را دانلود یا Clone کنید.
2. پوشه پروژه را در Android Studio باز کنید.
3. اجازه دهید Gradle Sync انجام شود.
4. اپ را روی شبیه‌ساز یا گوشی اجرا کنید.

## ساخت APK با GitHub Actions

1. یک Repository جدید در GitHub بسازید.
2. همه فایل‌های این پوشه را داخل Repository آپلود کنید.
3. وارد تب **Actions** شوید.
4. Workflow با نام **Build Android APK** را اجرا کنید، یا یک Commit جدید Push کنید.
5. بعد از پایان Build، فایل APK از بخش Artifacts با نام `LeatherCalculator-debug-apk` قابل دانلود است.

## تغییر نام پکیج

نام پکیج فعلی:

```text
com.fifers.leathercalculator
```

برای تغییر آن، مقدارهای زیر را ویرایش کنید:

- `app/build.gradle` بخش `namespace` و `applicationId`
- مسیر و خط اول فایل‌های `MainActivity.java` و `MonthlyResultsActivity.java`
- نام Activityها در `app/src/main/AndroidManifest.xml`

- نمایش جمع مابه‌التفاوت بر اساس ماه و محصول فیلترشده در صفحه سوابق ماهانه

## طراحی رابط کاربری نسخه جدید

- رابط RTL کارت‌محور با رنگ‌بندی گرم و متناسب با حوزه چرم
- هدر حرفه‌ای و دسترسی سریع به سوابق ماهانه
- فیلدها، دکمه‌ها و منوهای انتخاب با طراحی یکدست و خواناتر
- نمایش نتیجه محاسبه به‌صورت کارت برجسته با نشان رنگی وضعیت برش
- نمایش سوابق در کارت‌های مستقل با تفکیک بصری «برش خوب» و «برش بد»
- کارت خلاصه در صفحه گزارش برای جمع مابه‌التفاوت، تعداد رکورد و وضعیت مجموع


## نسخه UI/UX حرفه‌ای (Premium Dashboard)

- داشبورد اصلی با هدر مدیریتی، شمارنده محصولات و تعداد محاسبات ذخیره‌شده
- مسیر کاربری مرحله‌بندی‌شده برای محاسبه سریع و مدیریت کاتالوگ
- کارت نتیجه با نمایش برجسته مابه‌التفاوت، استاندارد کل و مصرف واقعی
- ورودی‌های focus-state، کارت‌ها و دکمه‌های یکپارچه با زبان بصری متناسب با چرم
- صفحه گزارش پیشرفته با جمع مابه‌التفاوت، مجموع مصرف استاندارد/واقعی و شمارنده برش خوب/بد برای فیلتر جاری
- حفظ کامل فیلتر ماه، جستجوی محصول، ذخیره سوابق و ساخت APK با GitHub Actions
